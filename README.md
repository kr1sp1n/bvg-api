BVG API
===================

An inofficial API to the BVG (Berlin Transport Services) written in Coffeescript.


Install
-------------------

    git clone git@github.com:kr1sp1n/bvg-api.git
    cd bvg-api
    npm install

Run
-------------------

    npm start


Develop
-------------------

Watch for changes and restart server automatically:

    gulp develop
